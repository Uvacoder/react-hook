# react-hooks

React hooks with meaningful examples.

#### Pre Requisite

You must have yarn installed before setting up this project.

-   **yarn**

#### Stack

-   Package Manager: **yarn**
-   Bundler: **Parcel**
-   Transpiler: **Babel**
-   UI: **React.JS**

#### Running

To install dependencies:

-   `yarn install`

To run the dev environment:

-   `yarn run dev`

To build:

-   `yarn run build`
